**MamieJago** itu platform e-commerce buat makanan siap saji: *mie jebew, wonton, dimsum*, tapi tampilannya **elegan, modern, dan gampang banget dipakai**.  
Dibangun pake **Node.js + Express + MongoDB Atlas + Tailwind CSS + Vanilla JS**, lengkap sama **keranjang belanja, checkout, invoice otomatis, QRIS Pakasir, dan admin panel aman**.

- **Sistem Review** ‚Äì Pengguna bisa kasih rating & komentar, admin bisa moderasi.
